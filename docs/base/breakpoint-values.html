Breakpoint Values

Sandbox comes with 3 responsive breakpoints out of the box, as shown in the table.

If you're customizing sandbox, these breakpoint ranges can easily be changed by plugging in your own desired values into the related sass variables and running a build.

Breakpoint Range Range in pixels Suffix
small ≥ 0rem && ≤ 48rem ≥ 0px && ≤ 768px -sm
medium ≥ 48.0625rem && ≤ 64rem ≥ 769px && ≤ 1024px -md
large ≥ 64.0625rem ≥ 1025px -lg
