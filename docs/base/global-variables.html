Global Variables

Description

If you're customizing sandbox and want to quickly change the look of it, the global variables file is the place to start. Here you can set the font stack that is used, default color, and other settings used throughout the framework.

Identifier Value
$FONT_STACK -apple-system, BlinkMacSystemFont, Roboto, "Helvetica Neue", Helvetica, Arial, sans-serif
$FONT_COLOR $alpha-black
$BORDER_COLOR $medium-white
$BORDER_ACTIVE_COLOR $dark-white
$GRID_SPACING 1.2rem
$RADIUS_ROUND 200em
$DISABLED_OPACITY 0.5
$BUTTON_BORDER_RADIUS 0.1875em
$CONTROL_HEIGHT 2.25em
$CONTROL_SHADOW_SPREAD 0.1875em
$BODY_BACKGROUND_COLOR rgb(252, 252, 252)
