Description

Sandbox uses a font scale of 1.3333 with a base font-size of 1.6rem. The font-size classes will override the font-size of any element they are used on.

Breakpoints

The font-size classes are responsive. Simply plug one of the three breakpoints into [breakpoint] to have that font-size become active during that breakpoint. Reduce or expand the browser width to see it in action.

  1. <button class="button block-center info h1-sm h3-md p-alt-lg">
    1. loading
    2. <span class="loading"></span>
  2. </button>

Heading elements

The heading elements, as well as the p element, have the related font-size already applied to them, as well as a line-height of 1.5.
