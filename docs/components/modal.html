Class overview

  • div.pseudo-body
  • div.pseudo-body.modal-backdrop
  • div.modal
  • body.pseudo-body-present

Description

A modal is an overlay that can be used to obscure the rest of the body content, and instead focus on something else.

The div.pseudo-body can be thought of as a body element which lies above the rest of the content on the page. You can put whatever content you want inside of it, as if it were the actual body element. The div.pseudo-body is positioned fixed, and has its z-index set to 55.

All that the .modal-backdrop class does for the div.pseudo-body is set a background-color.

The div.modal is like a blank canvas for you to fill with whatever content that you want. Use the div.modal (or other components) in conjunction with the grid column classes and the positioning classes for more control.

Using the .pseudo-body-present class on the body element makes it so the user cannot scroll the body element.

Markup

It's best to have the div.pseudo-body as the first child of the body element, in an effort to avoid z-index issues with other elements.

Demonstration

Structure

  1. <div class="pseudo-body modal-backdrop">
    1. <div class="absolute center-center unit-10-sm unit-9-md unit-8-lg">
      1. <div class="modal">
        1. <!-- your content here -->
      2. </div>
    2. </div>
  2. </div>

Implementation

Remember that Sandbox does not come with any JavaScript, so that you can implement your way. However, the rudimentary implementation used for this page's modal example is provided below:

  1. const pseudoBody = document.querySelector("#pseudoBody");
  2. const togglePseudoBody = document.querySelector("#togglePseudoBody");
  3. togglePseudoBody.addEventListener("click", function(e){
    1. document.body.classList.toggle("pseudo-body-present");
    2. pseudoBody.classList.toggle("absent");
  4. });
  5. pseudoBody.addEventListener("click", function(e){
    1. if (e.target.classList.contains("pseudo-body")){
      1. document.body.classList.toggle("pseudo-body-present");
      2. pseudoBody.classList.toggle("absent");
    2. }
  6. });
