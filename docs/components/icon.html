Class overview

  • span.icon

Description

The span.icon is a wrapper to be used around an icon, if you want to give it some additional width. This can help normalize the space between an icon and text.

Font Awesome

The examples below use the Font Awesome icon library. If you want to try the examples yourself, ensure that you've linked in the Font Awesome cdn.

Font-size scaling

While you can apply any of the font-size classes to a span.icon to scale it, it's intended to be scaled by the font-size it inherits from its parent element.
