Description

A dropdown element which can house navigational links or further nested dropdowns. The ul.dropdown has its position property set to absolute, meaning that it will be placed in relation to its most recently positioned ancenstor. Keep in mind how this can impact your layout horizontally.

The a.dropdown-a is written to be used within a li.dropdown-item.

The li.dropdown-label is used convey information without being clickable.

Nested dropdowns

A ul.dropdown can be nested inside of a li.dropdown-item. Remember to set the position property on the desired li.dropdown-item since the ul.dropdown is positioned based on its most recently positioned ancestor.
