Description

Use a figure with one of the .image- classes if you want to set aside space for an img element of that dimension (in pixels) squared, before it actually loads in, or if it even loads at all. The img element used within will be sized to that specified dimension.

Structure

  1. <figure class="image-16">
    1. <img src="" alt="an image">
  2. </figure>
  3. <figure class="image-32">
    1. <img src="" alt="an image">
  4. </figure>
  5. <figure class="image-64">
    1. <img src="" alt="an image">
  6. </figure>
  7. <figure class="image-128">
    1. <img src="" alt="an image">
  8. </figure>

Rounded

Round the img element by using the .rounded class on it.
