Description

A span.arrow is a mark which is typically found on popovers, but can be used within other components as well. Since its position property is set to absolute, it will be placed in relation to its most recently positioned ancestor.

Structure

The examples below build on a navbar and popover example.

Reusable

The span.arrow can be used on components other than the div.popover, such as the div.message-header and div.message-body. The span.arrow inherits both the background-color and the border-color properties from its parent element.
