Active

Use the .active class on a li.list-item or an li.list-label to indicate that that element is active.

Using the .indent-sublists class on a ul.list-menu will indent every ul.list-menu that is nested within it.

Theme classes

Apply either of the above mentioned theme classes to a ul.list-menu to have that theme's styles applied. Setting a theme class on the topmost ul.list-menu will style any ul.list-menu elements nested within it.

In this case the theme classes refer to what type of colored background the element is styled to be used on.

  • light
  • item 1
  • item 2
  1. <ul class="list-menu theme-light">
    1. <li class="list-label">
      1. <a href="" class="list-a">light</a>
    2. </li>
    3. <li class="list-item">
      1. <a href="" class="list-a">item 1</a>
    4. </li>
    5. <li class="list-item">
      1. <a href="" class="list-a">item 2</a>
    6. </li>
  2. </ul>
  • dark
  • item 1
  • item 2
  1. <ul class="list-menu theme-dark">
