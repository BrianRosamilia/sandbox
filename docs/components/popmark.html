Description

A span.popmark is a mark which is typically found on popovers, but can be used within other components as well. Since its position property is set to absolute, it will be placed in relation to its most recently positioned ancestor.

Reusable

The span.popmark can be used on components other than the div.popover, such as the div.message-header and div.message-body. The span.popmark inherits both the background-color and the border-color properties from its parent element.
