Class overview

  • div.progress-bar
  • div.progress

Description

A progress bar which can be used to indicate completion status.

Structure

  1. <div class="progress-bar">
    1. <div class="progress" style="width: 65%;"></div>
  2. </div>

Utilities

The div.progress is just a block level element with a background-color, that is floated to the left. Use some of the utility classes to control the placement of content within the div.progress.

  1. <div class="progress-bar">
    1. <div class="progress relative" style="width: 20%;">
      1. <span class="loading h6 absolute center-center"></span>
    2. </div>
  2. </div>
  1. <div class="progress-bar">
    1. <div class="progress flex flex-center flex-cross-center" style="width: 35%;">
      1. <span class="loading h6"></span>
    2. </div>
  2. </div>

Multiple

You can put multiple div.progress elements within a single div.progress-bar.

  1. <div class="progress-bar">
    1. <div class="progress theme-primary" style="width: 25%;"></div>
    2. <div class="progress theme-caution" style="width: 25%;"></div>
    3. <div class="progress theme-info" style="width: 50%;"></div>
  2. </div>

Theme classes

Apply any of the above mentioned theme classes to the div.progress to have that themes styles applied.

  1. <div class="progress-bar">
    1. <div class="progress theme-primary" style="width: 15%;"></div>
  2. </div>
  3. <div class="progress-bar">
    1. <div class="progress theme-info" style="width: 30%;"></div>
  4. </div>
  5. <div class="progress-bar">
    1. <div class="progress theme-caution" style="width: 45%;"></div>
  6. </div>
  7. <div class="progress-bar">
    1. <div class="progress theme-danger" style="width: 60%;"></div>
  8. </div>
  9. <div class="progress-bar">
    1. <div class="progress theme-light" style="width: 75%;"></div>
  10. </div>
  11. <div class="progress-bar">
    1. <div class="progress theme-dark" style="width: 90%;"></div>
  12. </div>

    1. <div class="progress" style="width: 25%"></div>
