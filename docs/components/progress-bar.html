Progress-Bar

displayunchanged positionunchanged z-indexunchanged font-size scalingyes breakpointsno

Class list

  • .progress-bar
  • .progress

Description

Use a .progress-bar to visually indicate how much .progress has been made.

Structure

<div class="progress-bar">
    <div class="progress" style="width: 65%;"></div>
</div>

Multiple

Any number of .progress elements can be put inside of a .progress-bar.

<div class="progress-bar">
    <div class="progress" style="width: 20%;"></div>
    <div class="progress primary" style="width: 30%;"></div>
    <div class="progress info" style="width: 40%;"></div>
</div>

Font size scaling

Use the font size utility classes on a .progress-bar to change its size.

<div class="progress-bar h6 mb-half">
    <div class="progress" style="width: 20%;"></div>
</div>
<div class="progress-bar mb-half">
    <div class="progress" style="width: 20%;"></div>
</div>
<div class="progress-bar h2">
    <div class="progress" style="width: 20%;"></div>
</div>

Themes

Use any of the provided theme classes on a .progress to have that themes styles applied.

<div class="progress-bar">
    <div class="progress primary" style="width: 15%"></div>
    <div class="progress info" style="width: 15%"></div>
    <div class="progress caution" style="width: 15%"></div>
    <div class="progress danger" style="width: 15%"></div>
    <div class="progress light" style="width: 15%"></div>
    <div class="progress dark" style="width: 15%"></div>
</div>
