Class overview

  • ul.tabs
  • li.tab

  • a.tab-a

Description

Use a ul.tabs to group together a collection of elements used for navigation. For additional control, combine the ul.tabs with the flexbox utility classes.

Use the .active class on a li.tab to indicate the active tab.
