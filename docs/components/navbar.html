  • li.navbar-item

Description

The div.navbar is very light-handed in its styling, allowing you to use the utility classes for the specific layout that you want. Apply one of your own background-color classes to the div.navbar for the desired look.

The li.navbar-item is the only element within the div.navbar that has a display property set. Its display property is set to inline-block.

Fixed top

Use the .fixed-top class on a div.navbar in order to have its display set to fixed, and be aligned to the top of the viewport.

Theme classes

Use the .theme-dark class on a div.navbar to have that theme's styles applied.

In this case the theme class refers to the type of colored background that the element is styled to be used on.
