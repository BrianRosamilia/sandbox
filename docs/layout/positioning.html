Description

Use the positioning classes to set an element's position property to the respective value.

Positioning presets

Use the .[x]-[y] classes in conjunction with one of the positioning classes (such as .fixed, or .absolute) in order to position an element. The .[x]-[y] classes use preset top, bottom, left and right values to do this.

The values that can be plugged into [x] are: left, center, right.

The values that can be plugged into [y] are: top, center, bottom.
