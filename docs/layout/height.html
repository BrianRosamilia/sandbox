Description

The height classes use a fraction notation to set the height property of an element to a percentage value, based on the height of that element's parent. Note that the element must be able to have height and width properties set on it (such as block or inline-block elements).

For example, .h-1-2 would set an element's height to 50% of its parent element.

Set Height

In order for the height classes to work on an element, the parent element must have a set height value.

Responsive height

The height classes are responsive. Simply plug in a desired breakpoint into [breakpoint].

  1. <div class="banner h1">
