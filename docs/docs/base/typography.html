Description

Sandbox's main draw is that the majority of its components can be scaled by using font-size classes on them, which are also breakpoint responsive.

Typography and color

The .t class stands for typography. This class sets the font-family, color (to Sandbox's $black variable) and a line-height. The .t class can be used on components to style the text within them, such as the div.stamp-bar component.

stamp bar
  1. <div class="stamp-bar t">
    1. <span class="stamp theme-light"></span>
    2. <span class="stamp theme-dark"></span>
  2. </div>
