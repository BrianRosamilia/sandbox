Description

The spacing utility classes have four levels which can be plugged into [#]. The levels are 1, 2, 3, 4. The value of each level is a fraction of the $spacing variable that is set, which has a value of 4.8rem.

For example, the value used by .mgn-1 would be 1.2rem, since 1/4 * 4.8 is equal to 1.2.

Responsive spacing

The spacing utility classes are responsive. Simply append -[breakpoint], where [breakpoint] is the name of the breakpoint, onto the end of the spacing utility class. Drag the width of your browser to see the example.
