Description

Sandbox offers a selection of Flexbox functionality as utility classes. For an in depth look at what Flexbox is about, check out the article: A Complete Guide to Flexbox.

The gist of the article is that Flexbox has settings for two types of elements, a flex container, and flex items (child elements of a flex container). Sandbox provides utility classes for the flex container only, but if you wanted to, you could expand with your own utility classes.

Usage

Use either the .flex or .flex-row-wrap classes to specify a flex container on a block level element.

The .flex- classes are used to indicate what to do with nested items along the main axis.

The .flex-cross- classes are used to indicate what to do with the nested items along the cross axis (the axis forming a 90 degree angle with the main axis).

  1. <div class="flex flex-around flex-cross-center">
    1. <h5 class="t h5 text-white">text</h5>
    2. <button class="button h3">button</button>
    3. <div class="demo-box demo-mred"></div>
  2. </div>
