Description

Use the display utility classes to set the display property to the respective value. Remember that setting the display property on an element of a specific type does not actually change the type, only what it's displayed as.

For example, doing span.block would only display the span element as a block level element. You would still be unable to nest block level elements inside of it.

Absent

The .absent class is used to set the display property to none. This removes the element from the flow of the page, as if it does not exist.

The .absent class is responsive. Simply plug in the desired breakpoint into [breakpoint].

Invisible

The .invisible class is used to set the visibility property to hidden. This still includes the element within the flow of the page, but only makes it so that it cannot be seen.

The .invisible class is responsive. Simply plug in the desired breakpoint into [breakpoint].
