Class overview

  • .to-right

  • .clear-right
  • .clear-both

Description

Use the .to-* classes to float an element to either the left or right. Use the .clear-* classes on an element that comes after a floated element to have it cleared. Use the .no-float class to set an element's float property to none.

For a more in depth look at what floating is all about, check out this article: CSS Float and Clear.
