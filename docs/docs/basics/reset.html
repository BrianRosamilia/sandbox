Reset

Specs

breakpoints no
theme classes no
font-size scaling no
size class scaling no
uses grid within no
variables no
display set for resetting
position set no

Description

Sandbox has its own reset.scss file which is used to reduce styling inconsistencies between browsers.

Most notably you'll see that the root element, html, has its font-size property set to 62.5%. Here's why:

Most browsers have a default font-size of 16px. Since Sandbox uses rem units, the default 1rem would be equal to 16px.

Working in units of 16 by default is not very easy. It'd be much easier to work in units of 10.

62.5% of 16 is equal to 10

Now when we use 1rem it will be equal to 10px, which makes it easier when choosing what rem value we want to work with.

Also note that html sets its height to 100% (of the viewport) and body sets its height to 100% (of html). This comes into play when using the height classes.

  1. /* inside reset.scss */
  2. html {
    1. font-size: 62.5%;
    2. height: 100%;
  3. }
  4. body {
    1. line-height: 1;
    2. height: 100%;
  5. }
