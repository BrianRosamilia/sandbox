Typography

The .t class stands for typography. This class sets the font-family using a native font stack, and also sets a line-height. This makes text easier to read when spanning multiple lines.

The .tc class stands for typography color, and is an extension of the .t class. The difference is that .tc class sets the color property, while .t class does not.
