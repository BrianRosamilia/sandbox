Description

The .t class stands for typography. This class sets the font-family using a native font stack, sets a line-height, and will inherit the color from its parent.

The .tc class is an extension of the .t class, which explicitly sets the color property to Sandbox's $black color variable.

Uses the .t class.

Inherits color from parent.

Uses the .tc class.

Sets color to $black.

Font Size

The font-size classes in Sandbox use a 1.3333 scale with a base size of 1.6rem.

Outliers

You may notice that there are two font-size classes provided which don't fit the scale (.p-alt and .h6), however they are useful font-sizes to have.

font-size class font-size
.huge-1 6.7rem
.huge-2 5.1rem
.h1 3.8rem
.h2 2.8rem
.h3 2.1rem
.h4, .p 1.6rem (base)
.p-alt 1.4rem (outlier)
.h5 1.2rem
.h6 1rem (outlier)

Responsive Font Size

The font-size classes are responsive. Simply plug in a desired breakpoint into [breakpoint]. Drag the width of your browser to see the example.
