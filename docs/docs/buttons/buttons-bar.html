Class overview

  • div.buttons-bar
  • div.buttons-bar.flex

Description

Use a div.buttons-bar to group multiple buttons together, edge to edge. Sandbox does this by floating each nested button to the left.

Structure

  1. <div class="buttons-bar">
    1. <button class="button">default</button>
    2. <button class="button theme-primary">primary</button>
    3. <button class="button theme-info">info</button>
    4. <button class="button theme-caution">caution</button>
    5. <button class="button theme-danger">danger</button>
    6. <button class="button theme-light">light</button>
    7. <button class="button theme-dark">dark</button>
  2. </div>

Flex

Using the .flex class on a div.buttons-bar designates it as a flex container, which will take up the fill width of the row. It will also make it so that all nested buttons take up an equal amount of the width.

  1. <div class="buttons-bar flex">
    1. <button class="button theme-primary">agree</button>
    2. <button class="button theme-info">meh</button>
    3. <button class="button theme-danger">disagree</button>
  2. </div>

Font Size Scaling

Apply any of the font-size classes to the div.buttons-bar to have it's children inherit the font-size specified.
