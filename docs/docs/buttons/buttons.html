Class overview

  • a.button
  • button.button
  • input[type="submit"].button
  • input[type="reset"].button
  • .button.wide

Description

Apply the .button class to the specified elements in order to have Sandbox's button styling applied.

Wide

Use the .wide class on one of the specified elements with the .button class to have its display set to block, and also take up the full width available.

Theme classes

Apply one of the .theme-* classes for the desired styling.

  1. <button class="button">default</button>
  2. <button class="button theme-primary">primary</button>
  3. <button class="button theme-info">info</button>
  4. <button class="button theme-caution">caution</button>
  5. <button class="button theme-danger">danger</button>
  6. <button class="button theme-light">light</button>
  7. <button class="button theme-dark">dark</button>

Font size scaling

Apply the font-size classes to adjust the size of the buttons.

  1. <button class="button h6">sizes</button>
  2. <button class="button h5">sizes</button>
  3. <button class="button p-alt">sizes</button>
  4. <button class="button h4">sizes</button>
  5. <button class="button h3">sizes</button>
  6. <button class="button h2">sizes</button>
  7. <button class="button h1">sizes</button>
  8. <button class="button huge-2">sizes</button>
  9. <button class="button huge-1">sizes</button>
