Class overview

  • div.container-buttons

Description

Use a div.container-buttons to arrange buttons using the grid column classes.

The container- classes

In Sandbox, any of the built in classes that start with container- are already clearfixed, and are intended to have the grid column classes used within them.

  1. <div class="container-buttons">

      1. <button class="button">button</button>

Font-Size scaling

Apply any of the font-size classes to div.container-buttons to have its children inherit the font-size specified.
