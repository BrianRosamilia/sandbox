Class overview

  • .button.outlined

Description

Apply the .outlined class to an appropriate element with the .button class, in order to have its outlined styling applied.

Theme classes

Apply one of the .theme-* classes for the desired styling.

  1. <button class="button outlined"> default </button>
  2. <button class="button theme-primary outlined"> primary </button>
  3. <button class="button theme-info outlined"> info </button>
  4. <button class="button theme-caution outlined"> caution </button>
  5. <button class="button theme-danger outlined"> danger </button>
  6. <button class="button theme-light outlined"> light </button>
  7. <button class="button theme-dark outlined"> dark </button>
