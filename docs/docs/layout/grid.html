Description

Sandbox provides a float based responsive grid system. Only two ingredients are required to build a responsive layout: a container, and grid columns within it.

All that div.container actually is, is a block level element that has clearfix code applied to it (to ensure that the floated elements within it don't exceed the container's boundaries). If you want to use some other block level element to house the grid columns, just apply the .clearfix class to it.

The grid columns are simply block level elements with some padding, and a set width (a percentage based the [#] you used) that are floated to the left.

CSS Selector

Any div element whose class list contains small- or medium- or large- or default- will be floated to the left, and have padding applied. Keep this in mind when expanding Sandbox with your own classes.

12 Unit Grid

Sandbox's grid is split into 12 equal units. Each [#] you plug in (from 1 - 12) sets the width property to a percentage based on that number out of 12. This width percentage is in relation to the parent element.

Pushing

The .push-[#], and .push-[#]-[breakpoint] classes are used to push a grid column to the right by the [#] specified ([#] being from 1 - 12). Sandbox does this by setting the margin-left property to the relevant percentage.

Stacking and Wrapping

Floated elements, such as the grid columns, will stack one next to the other, so long as there's enough room for them to fit within that line. If there's not enough room for the floated element to fit, it will wrap onto the next line.
