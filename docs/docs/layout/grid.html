Description

Sandbox uses a float based responsive grid system. Only two ingredients are required to build a responsive layout: a container, and grid columns within it.

Structure

small-12 medium-6 large-4

small-12 medium-6 large-4

small-12 medium-6 large-4

  1. <div class="container">
    1. <div class="small-12 medium-6 large-4"> <!-- your content here --> </div>
    2. <div class="small-12 medium-6 large-4"> <!-- your content here --> </div>
    3. <div class="small-12 medium-6 large-4"> <!-- your content here --> </div>
  2. </div>

Breakpoints

There are three breakpoints that can be plugged into [breakpoint]. These breakpoints are small, medium, and large. The table below shows when each breakpoint becomes active.

Name rem px
small <= 26.25rem <= 420px
medium >= 26.3125rem and <= 48rem >= 421px and <= 768px
large >= 48.0625rem >= 769px
default not responsive not responsive
