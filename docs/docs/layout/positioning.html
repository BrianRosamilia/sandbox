Description

Use the positioning utilities to set an element's position property to the respective value.

Positioning presets

The .position-[x]-[y] classes use preset top, bottom, left, and right values to position an element (absolute) in relation to its most recently positioned ancenstor.

The values that can be plugged into [x] are: left, center, right.

The values that can be plugged into [y] are: top, center, bottom.

  1. <div class="banner h1 relative">
    1. <div class="demo-box position-left-top"></div>
    2. <div class="demo-box position-left-center"></div>
    3. <div class="demo-box position-left-bottom"></div>
    4. <div class="demo-box position-center-top"></div>
    5. <div class="demo-box position-center-center"></div>
    6. <div class="demo-box position-center-bottom"></div>
    7. <div class="demo-box position-right-top"></div>
    8. <div class="demo-box position-right-center"></div>
    9. <div class="demo-box position-right-bottom"></div>
  2. </div>

CSS Selector

Any element whose class list contains position- will have its position property set to absolute. Keep this in mind when expanding Sandbox with your own classes.
