Class overview

  • label.checkbox
  • label.radio

Description

The label.checkbox and label.radio classes are wrappers for their respective input elements and some text.

Structure

  1. <label for="keys" class="checkbox">
    1. <input id="keys" type="checkbox">
    2. keys
  2. </label>
  3. <label for="phone" class="checkbox">
    1. <input id="phone" type="checkbox">
    2. phone
  4. </label>
  5. <label for="wallet" class="checkbox">
    1. <input id="wallet" type="checkbox">
    2. wallet
  6. </label>
  1. <label for="on" class="radio">
    1. <input id="on" type="radio" name="status" value="on">
    2. on
  2. </label>
  3. <label for="off" class="radio">
    1. <input id="off" type="radio" name="status" value="off">
    2. off
  4. </label>

Vertical

To have the label.checkbox and label.radio wrappers stack vertically, use the .block utility class on them.

  1. <label for="stacked" class="checkbox block">
    1. <input id="stacked" type="checkbox">
    2. stacked
  2. </label>
  3. <label for="vertically" class="radio block">
    1. <input id="vertically" type="radio">
    2. vertically
  4. </label>
