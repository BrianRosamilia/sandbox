Description

A div.field is an element that groups together a .label describing what the field is for, an input element (or textarea), and a .help for optional help text. A div.field, as well as input elements in general, are put inside of a form element.

Structure

Please enter your first name

  1. <div class="field">
    1. <label class="label">First name</label>
    2. <input type="text" class="textfield" name="firstName" placeholder="first name">
    3. <p class="help">Please enter your first name</p>
  2. </div>

Required

Using the .required class on a div.field puts an asterisk after the .label. Don't forget to use the required attribute on the input element within.
