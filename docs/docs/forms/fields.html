Description

A div.field is an element that groups together a p.label, describing the what the field is for, a control element (input, div.select, textarea.textarea, etc...), and a p.help for optional help text. A div.field, as well as other input elements, should live inside of a form element.

Note that the p.label, and the p.help have font-family and font-size styling preset.

Structure

First name

Enter your first name

  1. <form>
    1. <div class="field">
      1. <p class="label">First name</p>
      2. <input type="text" class="textfield" placeholder="John">
      3. <p class="help">Enter your first name</p>
    2. </div>
  2. </form>

Required field

Use the .required class on a div.field to have an asterisk placed after the p.label to indicate that the field is required. Don't forget to apply the required attribute to the control element itself.

Favorite primary color

Choose a color

  1. <form>
    1. <div class="field required">
      1. <p class="label">Favorite primary color</p>
      2. <div class="select">
        1. <select name="color" required>
          1. <option value="red">red</option>
          2. <option value="blue">blue</option>
          3. <option value="yellow">yellow</option>
        2. </select>
      3. </div>
      4. <p class="help">Choose a color</p>
    2. </div>
  2. </form>
