Class overview

  • div.container-controls

Description

Use a div.container-controls if you want to organize multiple controls onto the same line, using the grid columns to do so.

The container- classes

In Sandbox, any of the built in classes that start with container- are already clearfixed, and are intended to have the grid column classes used within them.

Structure

  1. <div class="container-controls">
    1. <div class="small-4 medium-6 large-3">
      1. <input type="text" class="textfield" placeholder="textfield">
    2. </div>
    3. <div class="small-4 medium-3 large-6">
      1. <textarea class="textarea" placeholder="textarea"></textarea>
    4. </div>
    5. <div class="small-4 medium-3 large-3">
      1. <div class="select">
        1. <select>
          1. <option value="select">select</option>
          2. <option value="choice">choice</option>
        2. </select>
      2. </div>
    6. </div>
  2. </div>
