Description

Sandbox uses div.select as a wrapper for the select element. This allows Sandbox to style the select element to be more consistent with other controls such as the input.textfield.

Structure

  1. <div class="select">
    1. <select>
      1. <option value="chicken">chicken</option>
      2. <option value="duck">duck</option>
      3. <option value="turkey">turkey</option>
    2. </select>
  2. </div>

Size classes

Unlike input.textfield, the div.select wrapper uses size-* classes to resize the element. The size-* classes are not responsive.

Recommended JavaScript

Since div.select is a wrapper for the select element, there is some recommended JavaScript to run in order to have the element behave as the other bordered controls do when they have focus.
