Class overview

  • div.select
  • div.select.multiple

Description

Sandbox uses a customized select element, wrapped by the div.select class. The div.select class adds some additional styling to complete the select element. This element is also styled to take up the full width available.

Structure

  1. <div class="select">
    1. <select>
      1. <option value="eggs">eggs</option>
      2. <option value="bacon">bacon</option>
      3. <option value="toast">toast</option>
    2. </select>
  2. </div>

Multiple

Apply the multiple attribute to the select element and use div.select.multiple in order to choose multiple options from the select element at once.

By setting the size attribute on the select element (that has the multiple attribute) you can specify how many options you want to be visible.

  1. <div class="select multiple">
    1. <select multiple size="3">
      1. <option value="hickory">hickory</option>
      2. <option value="applewood">applewood</option>
      3. <option value="maple">maple</option>
      4. <option value="smoked">smoked</option>
      5. <option value="peppered">peppered</option>
      6. <option value="turkey">turkey</option>
      7. <option value="canadian">canadian</option>
    2. </select>
  2. </div>

Font size scaling

Applying the font-size classes to div.select will scale the element, which will also match the size of a scaled input.textfield (except for the largest and smallest sizes).

  1. <div class="select h5">
    1. <select>
      1. <option value="h5">h5</option>
      2. <option value="t-bone">t-bone</option>
    2. </select>
  2. </div>
  3. <div class="select p-alt">
    1. <select>
      1. <option value="p-alt">p-alt</option>
      2. <option value="porterhouse">porterhouse</option>
    2. </select>
  4. </div>
  5. <div class="select h4">
    1. <select>
      1. <option value="h4">h4</option>
      2. <option value="ribeye">ribeye</option>
    2. </select>
  6. </div>
  7. <!-- etc... -->
