Class overview

  • div.field.container-field

Description

Use the .container-field class on a div.field when you want to create a horizontal form with the p.label on one side, and the control and p.help on the other.

The idea is that the div.field.container-field itself, becomes the housing for the grid columns to be used within.

Specific styling from the fields.scss file is also applied to the div.field.container-field.

The container- classes

In Sandbox, any of the built in classes that start with container- are already clearfixed (or set to flex in this case), and are intended to have the grid column classes used within them.
