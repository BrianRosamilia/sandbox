Class overview

  • ol.code
  • ol.code-nested
  • li.line
  • pre.code

Description

Use the ol.code element when you want to demonstrate a portion of code. Each individual line of code is written within an li.line element, and will have a line number generated for it.

The generated line number is positioned absolute, meaning that it is placed in relation to its most recently positioned ancestor (ol.code). The ol.code has its position set to relative.

HTML Character Entities

Be sure to replace the appropriate characters with their HTML character entity equivalent. A list of HTML entities can be found here.

Structure

  1. <span class="loading"></span>
  1. <ol class="code">
    1. <li class="line"><span class="loading"></span></li>
  2. </ol>

Nesting

Use the ol.code-nested within an ol.code to create a section of code which will be nested (via margin-left). You can continue nesting by using an ol.code-nested within other ol.code-nested elements.

  1. <div class="buttons-bar">
    1. <button class="button">one</button>
    2. <button class="button">two</button>
  2. </div>
  1. <ol class="code">
    1. <li class="line"><div class="buttons-bar"></li>
    2. <ol class="code-nested">
      1. <li class="line"><button class="button">one</button></li>
      2. <li class="line"><button class="button">two</button></li>
    3. </ol>
    4. <li class="line"></div></li>
  2. </ol>

Preformatted code

If you want to demonstrate a portion of code without having to write as many HTML tags, you can use the pre.code element. (Line numbers and nesting lines are not generated).

The formatting of the text placed within a pre element is preserved exactly as is.

this
  is
    preformatted
  1. <pre class="code">
  2. <!-- preformatted text here -->
  3. </pre>

Font-size scaling

Apply any of the font-size classes to an ol.code or a pre.code to scale the element and its children.

  1. <span class="loading"></span>
  1. <ol class="code h3">
    1. <li class="line"><span class="loading"></span></li>
  2. </ol>
