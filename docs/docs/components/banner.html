Class overview

  • div.banner

Description

A div.banner is just a block level element which sets a min-height and some padding. Combine it with some of your own background-color classes to make it stand out.

Structure

Lorem ipsum.

  1. <div class="banner demo-lgreen">
    1. <h3 class="text-white">Lorem ipsum.</h3>
  2. </div>

Font-size scaling

Use any of the font-size classes on a div.banner to scale the min-height value it uses.
