Class overview

  • div.modal-backdrop
  • div.modal
  • body.modal-present

Description

A modal is an overlay that can be used to obscure the rest of the body content, and instead focus on something else. The div.modal-backdrop is what does the obscuring, has its position set to fixed, and a z-index of 25.

Inside of the div.modal-backdrop is where you can use a div.modal, although in actuality you could put whatever you wanted inside of the div.modal-backdrop. The div.modal is like a blank canvas for you to fill with whatever content you see fit. Use a combination of the positioning and grid column classes in conjunction with the div.modal for fine tuned control of it.

Markup

It's best to have the div.modal-backdrop as the first child of the body element, in an effort to avoid z-index issues with other elements.

Implementation

Remember that Sandbox does not come with any JavaScript, so that you can implement the way you want to. The JS impelmentation used to launch this page's specific modal was:

  1. const modal = document.querySelector("#modal");
  2. const toggleModal = document.querySelector("#toggleModal");
  3. toggleModal.addEventListener("click", function(e){
    1. modal.classList.toggle("absent");
    2. document.body.classList.toggle("modal-present");
  4. });
  5. modal.addEventListener("click", function(e){
    1. if (e.target.className === "modal-backdrop"){
      1. modal.classList.toggle("absent");
      2. document.body.classList.toggle("modal-present");
    2. }
  6. });

Modal present

Using the .modal-present class on the body element makes it so that the user can no longer scroll the page. Consider using it while your modal is present.
