Class overview

  • button.close

Description

A close button which can be used pretty much anywhere.

Structure

  1. <button class="close"></button>

Theme classes

Apply the .theme-dark class to a button.close in order to have that theme's styles applied.
  1. <button class="close theme-dark"></button>

Font-size scaling

Apply any of the font-size classes to the button.close to scale it.

  1. <button class="close h6"></button>
  2. <button class="close h5"></button>
  3. <button class="close p-alt"></button>
  4. <button class="close h4"></button>
  5. <button class="close h3"></button>
  6. <button class="close h2"></button>
  7. <button class="close h1"></button>
  8. <button class="close huge-2"></button>
  9. <button class="close huge-1"></button>
