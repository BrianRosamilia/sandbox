Class overview

  • span.loading
  • span.loading.slow
  • span.loading.reversed
  • span.loading.paused

Description

A reusable, rotating loading icon.

Structure

The default border-color for the span.loading is $white, so a colored background is used for visibility.

  1. <span class="loading"></span>

Modifiers

Use the .slow, .reversed, or the .paused classes on a span.loading to modify its behavior.

  1. <span class="loading slow"></span>
  2. <span class="loading reversed"></span>
  3. <span class="loading paused"></span>
  4. <span class="loading slow reversed"></span>

Theme classes

Use the .theme-dark class on a span.loading to have that theme's styles applied.

  1. <span class="loading theme-dark"></span>

Font-size scaling

Use any of the font-size classes on a span.loading to scale the element.

  1. <span class="loading"></span>
  2. <span class="loading h3"></span>
  3. <span class="loading h2"></span>
  4. <span class="loading h1"></span>
  5. <span class="loading huge-2"></span>
  6. <span class="loading huge-1"></span>
