Description

A dropdown element which can house navigational links or further sub-dropdowns. The ul.dropdown has its position property set to absolute, meaning that it will be placed in relation to its most recently positioned ancenstor.

Structure

The examples below will build on top of a navbar example.

To test out the other classes which can be used on the ul.dropdown open up the dev tools and inspect the dropdown element. Open the .cls tab and apply the classes mentioned in the class overview.
