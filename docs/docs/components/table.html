Description

A styled table element which takes up the full width available to it.

Structure

Name Age
Hank 40
Hilda 62
Howard 58
  1. <table class="table">
    1. <thead>
      1. <tr>
        1. <th>Name</th>
        2. <th>Age</th>
      2. </tr>
    2. </thead>
    3. <tbody>
      1. <tr>
        1. <td>Hank</td>
        2. <td>40</td>
      2. </tr>
      3. <tr>
        1. <td>Hilda</td>

      4. </tr>

        1. <td>Howard</td>

Theme classes

Use any of the specified theme classes on a tr element to have that theme's styling applied.

Any tr with a theme class applied that is a child of a thead.row-hover, tbody.row-hover, or tfoot.row-hover will have theme related hover styling.
