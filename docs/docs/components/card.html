Description

A div.card is an element which can be used to organize different sections of content into a single place. Use a div.card-header, one or mulitple div.card-content, or a div.card-footer within the div.card to organize content into different sections.

You can put any content you want inside of a div.card-header, div.card-content, or div.card-footer.
