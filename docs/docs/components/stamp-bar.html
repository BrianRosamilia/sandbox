Class overview

  • div.stamp-bar
  • span.stamp.has-close

Description

A div.stamp-bar is a wrapper used around multiple stamps that you want to group together, such as those used in the Specs sections of this documentation.

Has close

If you want to use a button.close inside of a span.stamp that's within a div.stamp-bar, use the .has-close class on that span.stamp.

stamp
  1. <div class="stamp-bar t">
    1. <span class="stamp theme-dark">stamp</span>
    2. <span class="stamp theme-light has-close">
      1. <button class="close h3 no-background"></button>
    3. </span>
  2. </div>
