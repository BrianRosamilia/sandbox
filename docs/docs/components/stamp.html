Description

A span.stamp can be used to distinguish a specific piece of text from the rest of the text surrounding it, or even on its own.

By default, a span.stamp will inherit the font-size of its most recently set ancestor.

Structure

Lorem ipsum dolor sit amet, consectetur adipisicing.

Hello world
  1. <p class="t p">Lorem ipsum dolor sit amet, <span class="stamp">consectetur</span> adipisicing.</p>
  2. <span class="stamp t p">Hello world</span>

Round

Use the .round class on a span.stamp to round its edges even further.

stamp stamp
  1. <span class="stamp t p">stamp</span>
  2. <span class="stamp t p round">stamp</span>

Has link

Use the .has-link class on a span.stamp, and an a.stamp-a within it when you want to put a text link inside of a stamp.

link
  1. <span class="stamp t p has-link">
    1. <a href="" class="stamp-a">link</a>
  2. </span>

Theme classes

Use any of the specified theme classes on a span.stamp to have that theme's stlyes applied

primary info caution danger light dark

  1. <p class="t p">
    1. <span class="stamp theme-primary">primary</span>
    2. <span class="stamp theme-info">info</span>
    3. <span class="stamp theme-caution">caution</span>
    4. <span class="stamp theme-danger">danger</span>
    5. <span class="stamp theme-light">light</span>
    6. <span class="stamp theme-dark">dark</span>
  2. </p>

Font-size scaling
