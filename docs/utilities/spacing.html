Description

The spacing utility classes have four levels which can be plugged into [#]. The levels are 1, 2, 3, 4. The value of each level is a multiple of the $spacing variable that is set, which has a value of 1.2rem.

For example, the value used by .mgn-3 would be 3.6rem, since 3 * 1.2 is equal to 3.6.
