Description

Above, .pad stands for padding, .mgn stands for margin, .bdr stands for border, and .rds stands for radius (referring to the border-radius property). The t stands for top, b for bottom, l for left, and r for right.

The zeroing classes all follow a similar pattern. Using one of the classes without specifying a side, (t, b, l, r), will zero out all sides. If you want to zero out multiple specific sides, simply apply mutliple classes with the desired sides.
