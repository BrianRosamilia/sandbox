Class overview

  • div.buttons-bar
  • div.buttons-bar.flex

Description

Use a div.buttons-bar to group multiple buttons together, edge to edge. Sandbox does this by floating each nested button to the left.

Structure

  1. <div class="buttons-bar">
    1. <button class="button">default</button>
    2. <button class="button primary">primary</button>
    3. <button class="button info">info</button>
    4. <button class="button caution">caution</button>
  2. </div>

Flex

Using the .flex class on a div.buttons-bar designates it as a flex container, which will take up the full width available. It will also make it so that all nested buttons take up an equal amount of the width.

  1. <div class="buttons-bar flex">
    1. <button class="button danger">disagree</button>
    2. <button class="button light">meh</button>
    3. <button class="button primary">agree</button>
  2. </div>

No Wrap

Using the .flex class on a div.buttons-bar will make it so that it will not wrap onto a new line in any circumstance. Be mindful of this when creating responsive layouts.

Font Size Scaling

Apply any of the font-size classes to the div.buttons-bar to have its children inherit the font-size specified.

  1. <div class="buttons-bar h5">
    1. <button class="button">scaling</button>
    2. <button class="button">scaling</button>
    3. <button class="button">scaling</button>
  2. </div>
  3. <div class="buttons-bar h4">
