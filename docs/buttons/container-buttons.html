Class overview

  • div.grid-buttons

Description

Use a div.grid-buttons to arrange buttons using the grid column classes.

The grid- classes

In Sandbox, any of the built in classes that start with grid- are already clearfixed, and are intended to have the grid column classes used within them.

Font-Size scaling

Apply any of the font-size classes to div.grid-buttons to have its children inherit the font-size specified.
