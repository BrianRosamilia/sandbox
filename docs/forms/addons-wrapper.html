Description

Use a div.addons-wrapper to set some additional information to the left, right or both sides of an input.textfield or a div.select.

Structure

www.
.00
www.
.com
  1. <div class="addons-wrapper">
    1. <div class="left-addon">www.</div>
    2. <input type="text" class="textfield" placeholder="website.com">
  2. </div>
  3. <div class="addons-wrapper">
    1. <input type="number" class="textfield" placeholder="1">
    2. <div class="right-addon">.00</div>
  4. </div>
  5. <div class="addons-wrapper">
    1. <div class="left-addon">www.</div>
    2. <div class="select">
      1. <select>
        1. <option value="online">online</option>
        2. <option value="website">website</option>
        3. <option value="internet">internet</option>
      2. </select>
    3. </div>
    4. <div class="right-addon">.com</div>
  6. </div>

Has button

Using the .has-button class on a div.left-addon or a div.right-addon, and using one of the specified elements that can use .button inside of the addon, will fill that addon with the button.

  1. <div class="addons-wrapper">
    1. <div class="left-addon has-button">
      1. <button class="button theme-primary">select</button>
    2. </div>
    3. <input type="text" class="textfield" placeholder="something">
  2. </div>
  3. <div class="addons-wrapper">
    1. <input type="text" class="textfield" placeholder="search term">
    2. <div class="right-addon has-button">
      1. <button class="button theme-info">search</button>
    3. </div>
  4. </div>
  5. <div class="addons-wrapper">
    1. <div class="left-addon has-button">
      1. <button class="button">previous</button>
    2. </div>
    3. <div class="select">
      1. <select>
        1. <option value="choice">choice</option>
        2. <option value="other choice">other choice</option>
      2. </select>
    4. </div>
    5. <div class="right-addon has-button">
      1. <button class="button">next</button>
    6. </div>
  6. </div>
