Class overview

  • input.textfield

Description

The .textfield class is meant to be used on input element types which render as textfields, such as: email, number, password, search, tel, text, and url. Sandbox styles them to take up the full width available.

  1. <input type="email" placeholder="email" class="textfield">
  2. <input type="number" placeholder="number" class="textfield">
  3. <input type="password" placeholder="password" class="textfield">
  4. <input type="search" placeholder="search" class="textfield">
  5. <input type="tel" placeholder="tel" class="textfield">
  6. <input type="text" placeholder="text" class="textfield">
  7. <input type="url" placeholder="url" class="textfield">

Font-Size scaling

The suggested input element types with the .textfield class can be scaled by applying any of the font-size classes to them.

  1. <input type="text" placeholder="text" class="textfield h6">
  2. <input type="text" placeholder="text" class="textfield h5">
  3. <input type="text" placeholder="text" class="textfield p-alt">
  4. <input type="text" placeholder="text" class="textfield h4">
  5. <input type="text" placeholder="text" class="textfield h3">
  6. <input type="text" placeholder="text" class="textfield h2">
  7. <input type="text" placeholder="text" class="textfield h1">
