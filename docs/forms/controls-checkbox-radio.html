Class overview

  • label.checkbox
  • label.radio

Description

The label.checkbox and label.radio classes are wrappers for their respective input elements and some text.

Structure

  1. <label class="checkbox">
    1. <input type="checkbox" name="things" value="keys">
    2. keys
  2. </label>
  3. <label class="checkbox">
    1. <input type="checkbox" name="things" value="phone">
    2. phone
  4. </label>
  5. <label class="checkbox">
    1. <input type="checkbox" name="things" value="wallet">
    2. wallet
  6. </label>
  1. <label class="radio">
    1. <input type="radio" name="status" value="on">
    2. on
  2. </label>
  3. <label class="radio">
    1. <input type="radio" name="status" value="off">
    2. off
  4. </label>

Vertical

To have the label.checkbox and label.radio wrappers stack vertically, use the .block utility class on them.

  1. <label class="checkbox block">
    1. <input type="checkbox" name="stacked" value="stacked">
    2. stacked
  2. </label>
  3. <label class="radio block">
    1. <input type="radio" name="vertically" value="vertically">
    2. vertically
  4. </label>
