Icons-wrapper

Class overview

  • div.icons-wrapper
  • span.left-icon
  • span.right-icon

Description

The div.icons-wrapper is a wrapper meant to be used around an input.textfield, so that icons can be placed within the left and right sides of it.

The wrapper itself is positioned relative, while the span.left-icon and span.right-icon are positioned absolute.

Font Awesome

The examples below use the Font Awesome icon library. If you want to try the examples yourself ensure that you've linked in the Font Awesome cdn.

Structure

  1. <div class="icons-wrapper">
    1. <input type="number" class="textfield">
    2. <span class="left-icon">
      1. <i class="fa fa-dollar"></i>
    3. </span>
    4. <span class="right-icon">
      1. <i class="fa fa-bank"></i>
    5. </span>
  2. </div>

Markup

Ensure that the span.left-icon and span.right-icon come after the input.textfield element in the html. If they do not, the focus state will not behave as expected.

Font-size scaling

Apply any of the font-size classes to the div.icons-wrapper to have its children inherit the specified font size.
