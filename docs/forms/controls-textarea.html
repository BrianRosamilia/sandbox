Class overview

  • textarea.textarea
  • textarea.textarea.no-resize

Description

The textarea.textarea element shares styling with the input.textfield element, such as being styled to take up the full width that's available. It also sets restrictions for maximum and minimum height.

Structure

  1. <textarea class="textarea" placeholder="textarea"></textarea>

No-resize

Use the .no-resize class on a textarea.textrea to remove the resizing handle.

  1. <textarea class="textarea no-resize" placeholder="no resize"></textarea>

Font size scaling

Since the max/min-height and max/min-width of a textarea.textarea are preset, using a font-size class on it will mainly alter the border-radius and the font-size itself.

  1. <textarea class="textarea h6" placeholder="h6 textarea"></textarea>
  2. <textarea class="textarea h4" placeholder="regular textarea"></textarea>
  3. <textarea class="textarea h1" placeholder="h1 textarea"></textarea>
