Sandbox

Sandbox is a free and open source CSS framework.

It's a toolkit that makes building responsive layouts easy.

It provides a component library, buttons, forms and much more.

Get Sandbox Docs Github

Sandbox is

Scalable

Sandbox was built using relative units (primarily rem and em) so that components can be scaled by changing their font size.

Responsive

The grid classes, a variety of the utility classes, and the font-size classes are responsive (so that component and font sizes can be set per breakpoint).

Modular

Sandbox is written in SCSS and is structured so that you can choose what you'll use.

Customizable

Plug in your own values to Sandbox's SCSS variables, or hack the code to your own liking.

JavaScript Free

Sandbox compiles to a single CSS file, leaving you the freedom to implement your own JavaScript.

Lightweight

Sandbox under 65kb large, leaving plenty of room for custom expansion.
